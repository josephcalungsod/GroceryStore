# Grocery Store
We want user to be able to:
### (1) insert item
- Inserts new item
- *name / price / quantity*
### (2) update item
- Displays item 
- *name / price / quantity*
### (3) delete item
- Removes item. ( *if exists* )
### (4) display item
- Displays item
- *name / price / quantity*

#### potential primary/foreign keys 
- primary key = item
- foreign key = brand
#### example  
  - <b><u>table grocery_store</b></u>
    - <i>pk item: milk</i>
  - <b><u>table item_brand</b></u>
    - <i>fk brand: americas choice</i>
    - <i>fk brand: horizon</i>
